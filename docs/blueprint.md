# **App Name**: ESchoolBooks NeoX

## Core Features:

- Dashboard UI: Implement a dashboard UI for NeoX with a modern, academic-inspired design.
- File Upload System: Develop a file upload interface allowing users to submit previous year question papers and textbook PDFs.
- Exam Type Selection: Enable exam type selection, offering presets for Plus 2, PSC, NEET, JEE, and a custom option.
- AI-Powered Analysis Interface: Create an AI-powered analysis tool to process uploaded materials and identify key patterns.
- Exam Prediction Results Display: Design an interface that displays exam prediction results with confidence scores for each topic. LLM as tool may or may not choose to give the confidence score, at its discretion.
- Study Recommendations: Provide study recommendations based on the AI's analysis of uploaded materials.
- Progress UI Elements: Use lucide-react icon-based navigation for improved UI. Also create progress indicators and confidence meters.

## Style Guidelines:

- Primary color: Deep sky blue (#3498db) to convey trustworthiness and intelligence.
- Background color: Very light blue (#ecf0f1), nearly white to maintain a clean, academic look.
- Accent color: Emerald green (#2ecc71) for CTAs, progress indicators, and positive feedback.
- Body and headline font: 'Inter', a sans-serif font known for its modern, neutral, and readable style. This will be used for both headlines and body.
- Lucide React Icons will be used as this was specified.
- Card-based layout will be used to clearly separate exam information.
- Use subtle transitions and animations to enhance user engagement, such as progress animations during the analysis and uploading stages.
